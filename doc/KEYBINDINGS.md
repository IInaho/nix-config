# 快捷键

> **Mod 键**: 本地 TTY → `Super`；VM/嵌套窗口 → `Alt`

## Niri

### 常用

| 快捷键 | 功能 |
|--------|------|
| `Mod+Shift+/` | 快捷键覆盖层 |
| `Mod+T` | 终端 (kitty) |
| `Mod+D` | 应用启动器 (fuzzel) |
| `Mod+Shift+L` | 锁屏 (hyprlock) |
| `Mod+O` | 概览 |
| `Mod+Q` | 关闭窗口 |

### 焦点

| 快捷键 | 功能 |
|--------|------|
| `Mod+←↓↑→` | 焦点移动 |
| `Mod+Home/End` | 焦点移至列首/列尾 |

### 移动窗口

| 快捷键 | 功能 |
|--------|------|
| `Mod+Ctrl+←↓↑→` | 移动窗口/列 |
| `Mod+Ctrl+Home/End` | 移至首/末位 |

### 显示器

| 快捷键 | 功能 |
|--------|------|
| `Mod+Shift+←↓↑→` | 切换显示器焦点 |
| `Mod+Shift+Ctrl+←↓↑→` | 将列移至目标显示器 |

### 工作区

| 快捷键 | 功能 |
|--------|------|
| `Mod+1~9` | 切换工作区 |
| `Mod+Page_Down/Page_Up` | 上/下工作区 |
| `Mod+Ctrl+1~9` | 将列移至工作区 |
| `Mod+Ctrl+Page_Down/Page_Up` | 将列移至上/下工作区 |
| `Mod+Shift+Page_Down/Page_Up` | 移动工作区顺序 |

### 布局

| 快捷键 | 功能 |
|--------|------|
| `Mod+R` | 切换列宽预设 |
| `Mod+Shift+R` | 切换窗口高度预设 |
| `Mod+Ctrl+R` | 重置窗口高度 |
| `Mod+F` | 最大化列 |
| `Mod+Shift+F` | 全屏窗口 |
| `Mod+Ctrl+F` | 扩展至可用宽度 |
| `Mod+C` | 居中当前列 |
| `Mod+Ctrl+C` | 居中所有可见列 |
| `Mod+-/=` | 列宽 -/+10% |
| `Mod+Shift+-/=` | 窗口高度 -/+10% |

### 窗口操作

| 快捷键 | 功能 |
|--------|------|
| `Mod+V` | 浮动/平铺切换 |
| `Mod+Shift+V` | 浮动与平铺间切换焦点 |
| `Mod+W` | 列标签页显示模式 |
| `Mod+[` | 窗口吸入左侧列 / 弹出 |
| `Mod+]` | 窗口吸入右侧列 / 弹出 |
| `Mod+,` | 右侧窗口吸入当前列下方 |
| `Mod+.` | 当前列底部窗口弹出到右侧 |

### 截图 / 媒体 / 其他

| 快捷键 | 功能 |
|--------|------|
| `Mod+Shift+G` | 区域截图 → 剪贴板 |
| 音量键 | 音量 +/-10%，静音，麦克风静音 |
| 播放键 | 播放/暂停，停止，上一首，下一首 |
| 亮度键 | 亮度 +/-10% |
| `Mod+Escape` | 切换快捷键抑制 |
| `Mod+Shift+E` | 退出 Niri |
| `Ctrl+Alt+Delete` | 退出 Niri |
| `Mod+Shift+P` | 关闭显示器 |

### 滚轮

| 快捷键 | 功能 |
|--------|------|
| `Mod+滚轮↓↑` | 切换工作区 |
| `Mod+Ctrl+滚轮↓↑` | 将列移至工作区 |
| `Mod+滚轮←→` | 焦点左右移动 |
| `Mod+Ctrl+滚轮←→` | 列左右移动 |
| `Mod+Shift+滚轮↓↑` | 焦点左右移动 |
| `Mod+Ctrl+Shift+滚轮↓↑` | 列左右移动 |

| `<Space>fh` | 搜索帮助 |
| `Ctrl+h/j/k/l` | 窗口导航 |
