# Nixvim 快捷键指南

## Leader 键

**`Space` (空格键)** - 替代传统的 `\` 键，更符合人体工程学

---

## 快捷键总览

### 文件操作

| 快捷键 | 功能 |
|--------|------|
| `Space w` | 保存文件 (`:w`) |
| `Space q` | 退出 (`:q`) |

### 文件搜索 (Telescope)

| 快捷键 | 功能 |
|--------|------|
| `Space ff` | 查找文件 |
| `Space fg` | 全局搜索 |
| `Space fb` | 查找缓冲区 |
| `Space fh` | 帮助文档 |

### 目录树 (Neo-tree)

| 快捷键 | 功能 |
|--------|------|
| `Space e` | 打开/关闭目录树 |
| `h` | 折叠目录 |
| `l` | 展开目录 |
| `s` | 垂直分屏打开 |
| `i` | 水平分屏打开 |
| `o` | 打开文件 |

### LSP / 代码 (默认绑定)

| 快捷键 | 功能 |
|--------|------|
| `gd` | 跳转到定义 |
| `gr` | 查找引用 |
| `K` | 显示悬停信息 |

### 快速退出

| 快捷键 | 功能 |
|--------|------|
| `jk` | 退出插入模式 (更快) |
| `Esc` | 标准退出 |

---

## 窗口操作

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+h` | 切换左窗口 |
| `Ctrl+j` | 切换下窗口 |
| `Ctrl+k` | 切换上窗口 |
| `Ctrl+l` | 切换右窗口 |
| `Ctrl+w v` | 垂直分屏 |
| `Ctrl+w s` | 水平分屏 |
| `Ctrl+w q` | 关闭分屏 |
| `Ctrl+w =` | 重新平衡 |

---

## 移动

| 快捷键 | 功能 |
|--------|------|
| `h/j/k/l` | 左/下/上/右 |
| `w` | 下一个单词 |
| `b` | 上一个单词 |
| `e` | 单词结尾 |
| `0` | 行首 |
| `$` | 行尾 |
| `gg` | 文件开头 |
| `G` | 文件结尾 |

---

## 编辑

| 快捷键 | 功能 |
|--------|------|
| `i` | 插入模式 |
| `a` | 后插入 |
| `o` | 下方新行 |
| `O` | 上方新行 |
| `x` | 删除字符 |
| `dd` | 删除行 |
| `yy` | 复制行 |
| `p` | 粘贴 |
| `u` | 撤销 |
| `Ctrl+r` | 重做 |

---

## Git (gitsigns)

| 快捷键 | 功能 |
|--------|------|
| `]c` | 下一变更 |
| `[c` | 上一变更 |
| `gh` | 查看变更 |

---

## 模式

| 快捷键 | 功能 |
|--------|------|
| `i` | 插入模式 |
| `Esc` | 普通模式 |
| `v` | 可视(字符) |
| `V` | 可视(行) |
| `Ctrl+v` | 可视(块) |

---

## 常用命令

| 命令 | 功能 |
|------|------|
| `:wq` | 保存退出 |
| `:q!` | 强制退出 |
| `/pattern` | 搜索 |
| `n/N` | 下一个/上一个 |
| `:%s/a/b/g` | 全局替换 |

---

## 插件列表

- **tokyonight** - 主题 (night风格)
- **startupify** - 启动画面
- **neo-tree** - 目录树
- **vim-surround** - 快速操作周围符号
- **nvim-cmp** - 自动补全
- **gitsigns** - Git 状态显示
- **telescope** - 文件搜索/模糊查找
- **treesitter** - 语法高亮
- **lsp** - 语言服务器协议

## 主题切换

当前使用 tokyonight 主题，风格为 night（暗色）。
