# 快捷键配置说明文档

本文档汇总了本 NixOS 配置中的所有快捷键。

## 目录

- [Niri 窗口管理器快捷键](#niri-窗口管理器快捷键)
- [Neovim 编辑器快捷键](#neovim-编辑器快捷键)

---

## 键位说明

> **Mod 键**: 
> - 本地 TTY 运行: `Super`（Windows 键 / macOS Command 键）
> - VM/嵌套窗口中运行: `Alt`
>
> **Leader 键** (Neovim): `Space`（空格键）

## Niri 窗口管理器快捷键

### 基础操作

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Mod + Shift + /` | 显示快捷键覆盖层 | 弹出快捷键概览 |
| `Mod + T` | 打开终端 | 启动 kitty 终端 |
| `Mod + D` | 应用程序启动器 | 启动 fuzzel |
| `Mod + Shift + L` | 锁定屏幕 | 启动 swaylock |
| `Mod + Shift + S` | 启动/重启 orca | 屏幕阅读器 |

### 窗口焦点切换

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Mod + H` / `Mod + ←` | 焦点移到左侧窗口 | |
| `Mod + J` / `Mod + ↓` | 焦点移到下方窗口 | |
| `Mod + K` / `Mod + ↑` | 焦点移到上方窗口 | |
| `Mod + L` / `Mod + →` | 焦点移到右侧窗口 | |
| `Mod + Home` | 焦点移到列首窗口 | |
| `Mod + End` | 焦点移到列尾窗口 | |

### 窗口移动

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Mod + Ctrl + H` / `Mod + Ctrl + ←` | 将列向左移动 | |
| `Mod + Ctrl + J` / `Mod + Ctrl + ↓` | 将窗口向下移动 | |
| `Mod + Ctrl + K` / `Mod + Ctrl + ↑` | 将窗口向上移动 | |
| `Mod + Ctrl + L` / `Mod + Ctrl + →` | 将列向右移动 | |
| `Mod + Ctrl + Home` | 将列移至首位 | |
| `Mod + Ctrl + End` | 将列移至末位 | |

### 显示器间切换

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Mod + Shift + ←` | 切换到左显示器 | |
| `Mod + Shift + ↓` | 切换到下显示器 | |
| `Mod + Shift + ↑` | 切换到上显示器 | |
| `Mod + Shift + →` | 切换到右显示器 | |

### 窗口在显示器间移动

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Mod + Shift + Ctrl + ←` | 将列移至左显示器 | |
| `Mod + Shift + Ctrl + ↓` | 将列移至下显示器 | |
| `Mod + Shift + Ctrl + ↑` | 将列移至上显示器 | |
| `Mod + Shift + Ctrl + →` | 将列移至右显示器 | |

### 工作区操作

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Mod + 1 ~ 9` | 切换到工作区 1-9 | |
| `Mod + Page_Down` / `Mod + U` | 切换到下方工作区 | |
| `Mod + Page_Up` / `Mod + I` | 切换到上方工作区 | |
| `Mod + Ctrl + 1 ~ 9` | 将列移到工作区 1-9 | |
| `Mod + Ctrl + Page_Down` / `Mod + Ctrl + U` | 将列移到下方工作区 | |
| `Mod + Ctrl + Page_Up` / `Mod + Ctrl + I` | 将列移到上方工作区 | |
| `Mod + Shift + Page_Down` / `Mod + Shift + U` | 移动工作区向下 | |
| `Mod + Shift + Page_Up` / `Mod + Shift + I` | 移动工作区向上 | |

### 窗口布局调整

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Mod + R` | 切换预设列宽度 | 循环切换 1/3、1/2、2/3 宽度 |
| `Mod + Shift + R` | 切换预设窗口高度 | |
| `Mod + Ctrl + R` | 重置窗口高度 | |
| `Mod + F` | 最大化列 | |
| `Mod + Shift + F` | 全屏窗口 | |
| `Mod + Ctrl + F` | 扩展列至可用宽度 | |
| `Mod + C` | 居中当前列 | |
| `Mod + Ctrl + C` | 居中所有可见列 | |
| `Mod + -` | 减小列宽 10% | |
| `Mod + =` | 增大列宽 10% | |
| `Mod + Shift + -` | 减小窗口高度 10% | |
| `Mod + Shift + =` | 增大窗口高度 10% | |

### 窗口操作

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Mod + Q` | 关闭窗口 | |
| `Mod + V` | 切换窗口浮动/平铺 | |
| `Mod + Shift + V` | 在浮动和平铺间切换焦点 | |
| `Mod + W` | 切换列的标签页显示模式 | |
| `Mod + O` | 打开概览视图 | 显示所有工作区和窗口 |
| `Mod + [` | 窗口吸入左侧列 / 从左侧列弹出 | |
| `Mod + ]` | 窗口吸入右侧列 / 从右侧列弹出 | |
| `Mod + ,` | 将右侧窗口吸入当前列下方 | |
| `Mod + .` | 将当前列底部窗口弹出到右侧 | |

## Neovim 编辑器快捷键

### 通用快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `<Space>w` | 保存文件 | `:w` |

| `jk` | 退出插入模式 | 快速 Escape |

| 快捷键 | 功能 | 说明 |

| `<Space>ff` | 查找文件 | fuzzy find files |
| `<Space>fg` | 搜索文件内容 | live grep |
| `<Space>fb` | 搜索打开的缓冲区 | buffers |
| `<Space>fh` | 搜索帮助文档 | help tags |
