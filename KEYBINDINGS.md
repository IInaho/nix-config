## 键位说明

> **Mod 键**: 
> - 本地 TTY 运行: `Super`（Windows 键 / macOS Command 键）
> - VM/嵌套窗口中运行: `Alt`

## Niri 窗口管理器快捷键

### 常用操作

| 快捷键 | 功能 |
|--------|------|
| `Mod + Shift + /` | 显示快捷键覆盖层 |
| `Mod + T` | 打开终端 (kitty) |
| `Mod + D` | 应用程序启动器 (fuzzel) |
| `Mod + Shift + L` | 锁定屏幕 (hyprlock) |
| `Mod + O` | 概览视图 |
| `Mod + Q` | 关闭窗口 |

### 焦点移动

| 快捷键 | 功能 |
|--------|------|
| `Mod + ←` | 焦点移到左侧窗口 |
| `Mod + ↓` | 焦点移到下方窗口 |
| `Mod + ↑` | 焦点移到上方窗口 |
| `Mod + →` | 焦点移到右侧窗口 |
| `Mod + Home` | 焦点移到列首 |
| `Mod + End` | 焦点移到列尾 |

### 移动窗口/列

| 快捷键 | 功能 |
|--------|------|
| `Mod + Ctrl + ←` | 将列向左移动 |
| `Mod + Ctrl + ↓` | 将窗口向下移动 |
| `Mod + Ctrl + ↑` | 将窗口向上移动 |
| `Mod + Ctrl + →` | 将列向右移动 |
| `Mod + Ctrl + Home` | 将列移至首位 |
| `Mod + Ctrl + End` | 将列移至末位 |

### 显示器切换

| 快捷键 | 功能 |
|--------|------|
| `Mod + Shift + ←` | 切换到左显示器 |
| `Mod + Shift + ↓` | 切换到下显示器 |
| `Mod + Shift + ↑` | 切换到上显示器 |
| `Mod + Shift + →` | 切换到右显示器 |
| `Mod + Shift + Ctrl + ←` | 将列移至左显示器 |
| `Mod + Shift + Ctrl + ↓` | 将列移至下显示器 |
| `Mod + Shift + Ctrl + ↑` | 将列移至上显示器 |
| `Mod + Shift + Ctrl + →` | 将列移至右显示器 |

### 工作区

| 快捷键 | 功能 |
|--------|------|
| `Mod + 1 ~ 9` | 切换到工作区 1-9 |
| `Mod + Page_Down` | 切换到下方工作区 |
| `Mod + Page_Up` | 切换到上方工作区 |
| `Mod + Ctrl + 1 ~ 9` | 将列移到工作区 1-9 |
| `Mod + Ctrl + Page_Down` | 将列移到下方工作区 |
| `Mod + Ctrl + Page_Up` | 将列移到上方工作区 |
| `Mod + Shift + Page_Down` | 移动工作区向下 |
| `Mod + Shift + Page_Up` | 移动工作区向上 |

### 布局调整

| 快捷键 | 功能 |
|--------|------|
| `Mod + R` | 切换预设列宽度 |
| `Mod + Shift + R` | 切换预设窗口高度 |
| `Mod + Ctrl + R` | 重置窗口高度 |
| `Mod + F` | 最大化列 |
| `Mod + Shift + F` | 全屏窗口 |
| `Mod + Ctrl + F` | 扩展列至可用宽度 |
| `Mod + C` | 居中当前列 |
| `Mod + Ctrl + C` | 居中所有可见列 |
| `Mod + -` | 减小列宽 10% |
| `Mod + =` | 增大列宽 10% |
| `Mod + Shift + -` | 减小窗口高度 10% |
| `Mod + Shift + =` | 增大窗口高度 10% |

### 窗口操作

| 快捷键 | 功能 |
|--------|------|
| `Mod + V` | 切换窗口浮动/平铺 |
| `Mod + Shift + V` | 在浮动和平铺间切换焦点 |
| `Mod + W` | 切换列的标签页显示模式 |
| `Mod + [` | 窗口吸入左侧列 / 从左侧列弹出 |
| `Mod + ]` | 窗口吸入右侧列 / 从右侧列弹出 |
| `Mod + ,` | 将右侧窗口吸入当前列下方 |
| `Mod + .` | 将当前列底部窗口弹出到右侧 |

### 截图

| 快捷键 | 功能 |
|--------|------|
| `Mod + Shift + G` | 区域截图并复制到剪贴板 |

### 滚轮操作

| 快捷键 | 功能 |
|--------|------|
| `Mod + 滚轮 ↓` | 切换到下方工作区 |
| `Mod + 滚轮 ↑` | 切换到上方工作区 |
| `Mod + Ctrl + 滚轮 ↓` | 将列移到下方工作区 |
| `Mod + Ctrl + 滚轮 ↑` | 将列移到上方工作区 |
| `Mod + 滚轮 →` | 焦点移到右侧列 |
| `Mod + 滚轮 ←` | 焦点移到左侧列 |
| `Mod + Ctrl + 滚轮 →` | 将列向右移动 |
| `Mod + Ctrl + 滚轮 ←` | 将列向左移动 |
| `Mod + Shift + 滚轮 ↓` | 焦点移到右侧列 |
| `Mod + Shift + 滚轮 ↑` | 焦点移到左侧列 |
| `Mod + Ctrl + Shift + 滚轮 ↓` | 将列向右移动 |
| `Mod + Ctrl + Shift + 滚轮 ↑` | 将列向左移动 |

### 媒体键

| 快捷键 | 功能 |
|--------|------|
| `XF86AudioRaiseVolume` | 增加音量 10% |
| `XF86AudioLowerVolume` | 降低音量 10% |
| `XF86AudioMute` | 切换静音 |
| `XF86AudioMicMute` | 切换麦克风静音 |
| `XF86AudioPlay` | 播放/暂停媒体 |
| `XF86AudioStop` | 停止媒体 |
| `XF86AudioPrev` | 上一首 |
| `XF86AudioNext` | 下一首 |
| `XF86MonBrightnessUp` | 增加亮度 10% |
| `XF86MonBrightnessDown` | 降低亮度 10% |

### 其他

| 快捷键 | 功能 |
|--------|------|
| `Mod + Escape` | 切换键盘快捷键抑制 |
| `Mod + Shift + E` | 退出 Niri |
| `Ctrl + Alt + Delete` | 退出 Niri |
| `Mod + Shift + P` | 关闭显示器 |

## Neovim 编辑器快捷键

### 通用快捷键

| 快捷键 | 功能 |
|--------|------|
| `<Space>w` | 保存文件 |
| `<Space>q` | 退出文件 |
| `<Space>e` | 打开/关闭文件树 |
| `jk` | 退出插入模式 |

### Telescope 模糊搜索

| 快捷键 | 功能 |
|--------|------|
| `<Space>ff` | 查找文件 |
| `<Space>fg` | 搜索文件内容 |
| `<Space>fb` | 搜索打开的缓冲区 |
| `<Space>fh` | 搜索帮助文档 |

### 窗口导航

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + h` | 切换到左侧窗口 |
| `Ctrl + j` | 切换到下方窗口 |
| `Ctrl + k` | 切换到上方窗口 |
| `Ctrl + l` | 切换到右侧窗口 |
